# Ecommerce-Sales-Dashboard-Power-BI
## Project Overview
This project presents an interactive Power BI dashboard for analyzing e-commerce sales performance.

## Objectives
- Analyze sales trends
- Identify top performing products
- Understand regional sales distribution
- Track key revenue metrics

## Tools Used
- Power BI
- Excel / CSV dataset
- Data visualization techniques

## Key Insights
- Identified highest revenue generating categories
- Analyzed monthly sales growth
- Compared product performance across regions
